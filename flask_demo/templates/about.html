关于这个项目

这个 Flask 示例项目是为了帮助初学者理解 Flask Web 框架的基本概念而创建的。

🚀 项目特性

  • 简单易懂的代码结构
  • 包含常用的 Flask 功能演示
  • 响应式的 HTML 模板
  • 表单处理示例
  • REST API 示例

📁 项目结构

flask_demo/
├── app.py              # 主应用文件
└── templates/          # HTML 模板目录
    ├── base.html       # 基础模板
    ├── index.html      # 首页模板
    ├── about.html      # 关于页面模板
    ├── contact.html    # 联系表单模板
    ├── contact_success.html # 表单提交成功页面
    └── user_profile.html    # 用户资料页面模板

🛠️ 使用的技术

  • Flask - Python Web 框架
  • Jinja2 - 模板引擎
  • HTML/CSS - 前端样式
