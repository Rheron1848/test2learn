💡 动态路由解析

这个页面展示了 Flask 的动态路由功能：

1. 路由参数

@app.route('/user/<int:user_id>')
def user_profile(user_id):
    # user_id 是从URL中提取的整数参数

2. 参数类型

  • <int:user_id> - 整数类型参数
  • <string:name> - 字符串类型参数（默认）
  • <float:price> - 浮点数类型参数
  • <path:filepath> - 路径类型参数（包含斜杠）

3. URL 生成

# 在模板中生成URL
url_for('user_profile', user_id=1)
# 生成: /user/1

4. 错误处理

如果访问不存在的用户ID，应用会返回 404 错误。试试访问 /user/999
