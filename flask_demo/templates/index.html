📚 Flask 基础概念

Flask 是一个轻量级的 Python Web 框架，它提供了构建 Web 应用程序的基本工具和功能。

核心组件：

  • 应用实例: app = Flask(__name__)
  • 路由装饰器: @app.route('/path')
  • 视图函数: 处理请求并返回响应的函数
  • 模板: HTML文件，可以包含动态内容
